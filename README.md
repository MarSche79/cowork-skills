# Cowork Skills (Sanitized Export)

This repository contains a generalized, privacy-sanitized copy of custom Cowork skills for reuse.

All personal identifiers were replaced with placeholders (for example `<username>`, `CUSTOMER_NAME`, `<your-email@example.com>`), and machine-specific config files such as `design/config.json` were excluded.

## Import

1. Copy one or more folders from `skills/` into your local skills directory.
2. Open each `SKILL.md` and replace placeholders with values for your own environment.
3. Adjust paths and tool names where your setup differs.
